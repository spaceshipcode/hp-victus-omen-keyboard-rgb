# hp-victus-omen-keyboard-rgb
This repository allows you to easily control RGB keyboard backlighting on HP Victus/Omen laptops using Linux.

Since laptop manufacturers don't release applications for Linux, third-party tools are necessary.

This application provides the simple and easy interface required for control, installs the necessary dependencies (Python3, GTK4, libadwaita, DKMS, kernel headers), and performs the installation. It allows you to easily and freely control your keyboard backlighting via Linux.

While OpenRGB and its derivatives work for devices connected via USB, they don't work properly and easily with PCI-connected backlighting like on laptops such as Victus.

---
Installation :

In the terminal, navigate to the downloaded folder and run the command `bash setup.sh`. Once the installation is complete, it will appear as an application. If the icon doesn't appear, you can run the command `./kbd_backlight.py` in the terminal where you navigated to the folder.
